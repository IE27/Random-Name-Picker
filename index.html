    User Guide

    Comprehensive guide for using Random Name Picker:

    1. Enter Names:
      • Input your list of names in the provided text area.
      • Separate names with new lines, commas (,), or semicolons (;).
      • Example: John Doe, Jane Smith, Alice Johnson.
    2. Set List:
      • Click the "Set List" button to save your name list.
      • Verify your list before proceeding.
    3. Pick Winners:
      • Set the number of winners in the "Winners" field.
      • Click "Pick Random Name" to select winners randomly.
      • Selected names will appear in the "Selected Names" section.
    4. Additional Features:
      • Shuffle: Randomize the name list again
      • Reset: Clear all selected names
      • Export/Import: Save and load name lists
